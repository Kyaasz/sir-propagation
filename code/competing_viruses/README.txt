Ce répertoire contient les codes pour les simulations de virus concurrents. Il contient les fichiers suivants : 

- "animated_graph.py" : code pour l'animation de la simulation
- "app_competing.py" : code pour lancer l'application (interface graphique)
- "competing_propag.py" : code pour faire des simulations
- "graph_type.py" : code pour générer les différents types de graphes
