Ce répertoire contient les ressources sur le projet de recherche réalisé en stage au LIA sur l'étude de propagations d'épidémie dans des réseaux.

Il contient différents autres répertoires: 

1) Le répertoire "Rapports" contient trois rapports d'études : 
- "Rapport ML LIA" : Un premier sur la prédiction de métriques des simulations avec des outils de machine learning
- "Rapport Game Theory LIA" : Un deuxième sur l'étude d'un jeu compétitif entre un attaquant et un défenseur, l'attaquant cherchant à diffuser un virus
et le défenseur un antivirus, avec des outils de théorie des jeux.
- "compte-rendu jeu minimisation cout honeypots" : un troisième rapport sur l'étude d'un jeu compétitif entre un attaquant et un défenseur, mais
cette fois le défenseur peut influer sur les probabilités de changements d'état et la mise en place de honeypots.

2) Le répertoire "Manuels utilisateur simulateurs" contient 3 manuels : 
Ce répertoire contient les 3 manuels des 3 simulateurs disponibles
- "Mode d'emploi simulateur" : Mode d'emploi du simulateur de base, c'est-à-dire celui avec
la mise en place de honeypos. 
- "Mode d'emploi simulateur virus antivirus" : Mode d'emploi du simulateur avec la diffusion
d'un virus et la diffusion d'un antivirus
- "Mode d'emploi simulateur virus concurrents" : Mode d'emploi du simulateur de propagations
de deux virus concurrents.

3) Le répertoire "documents de travail" contient divers documents retraçant les résultats 
obtenus avant la rédaction des rapports.

4) Le répertoire "code" contient les différents codes. Un fichier README y est présent
pour renseigner sur les différents répertoires et fichiers présents dans le dossier.
